# Spin the Wheel – Youth Programme Q&A Website

This repository contains a web-based interactive **Spin the Wheel** application created for a youth programme activity. The website allows participants to spin a virtual wheel that randomly selects questions, creating a fun and engaging way to encourage discussion, participation, and learning.

## Project Overview

The goal of this project was to build a simple, interactive web tool that could be used during youth programme sessions to randomly generate questions for participants. The spinning wheel format makes the activity more dynamic and encourages involvement from everyone in the group.

Participants spin the wheel, and the segment selected reveals a question or prompt for discussion.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  

## Features

- Interactive spinning wheel interface  
- Randomized question selection  
- Simple and user-friendly design  
- Responsive layout suitable for different screen sizes  

## Use Case

This application was developed to support youth engagement activities by making question-and-answer sessions more interactive and enjoyable. It can be used in workshops, youth meetings, team-building activities, or educational programmes.

## Purpose of the Project

This project demonstrates:
- Front-end web development skills
- JavaScript-based interactivity
- UI design for engagement and usability

## Possible Future Improvements

Potential enhancements include:

- Adding more question categories
- Allowing administrators to add custom questions
- Sound effects for the spinning animation
- Score tracking for quiz-style sessions
